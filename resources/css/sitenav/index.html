This is a CSS experiment with CSS Grid and CSS Variables, spawned by a screenshot tweeted by Matt Brailsford.

When I see something like that, my mind goes into “Oh, I wonder how I’d do that…” mode, and I might as well give it a try, ‘coz it won’t let me forget about it :-) Finally had some time on my hands to do it.

This is a work in progress — you can follow along on GitHub: github.com/greystate/sitemap-test/. Matt’s version is also on GitHub: github.com/mattbrailsford/css-sitemap/.
